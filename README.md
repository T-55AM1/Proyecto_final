# 🤖 Chatbot con Deep Learning

Chatbot conversacional basado en redes neuronales (LSTM) desarrollado con **Python** y **TensorFlow**. El modelo ya se encuentra entrenado y listo para usarse.

## 🚀 Cómo usar el Chatbot en vivo 🎮

Ejecuta estos dos comandos en la terminal para empezar a chatear de inmediato:

### 1. Instalar librerías básicas
```bash
pip install pandas tensorflow
```

### 2. Lanzar el Chatbot 
```bash
python 04_chat_en_vivo.py
```

---

## 💬 Instrucciones de Uso
* **Hablar**: Escribe frases como `hola`, `como estas?` o `mi computadora no enciende` y presiona **Enter**.
* **Cerrar**: Escribe la palabra **`salir`** y presiona **Enter** para apagar el chat.
